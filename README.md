# *MemoLoom*

*A structured creativity tool for evolving ideas, comparing versions, and uncovering clearer thinking.*

MemoLoom helps designers, writers, and builders transform scattered notes into organized idea threads.
It visualizes how ideas evolve, compares versions side-by-side, and uses AI to spark new creative directions.
